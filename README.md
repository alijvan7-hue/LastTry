# 🤖 آپلودر تریاک - ربات تلگرامی

ربات تلگرامی حرفه‌ای برای آپلود و اشتراک‌گذاری فایل از طریق لینک اختصاصی.

## ✨ امکانات

### 👤 کاربران
- 📥 دریافت فایل از طریق لینک اختصاصی
- 🔒 عضویت اجباری در کانال‌ها قبل از دریافت فایل
- 📞 تماس با پشتیبانی

### 👑 مدیریت
- 📊 بررسی وضعیت (آمار کامل ربات)
- 📢 پیام همگانی به تمام کاربران
- 📂 مدیریت فایل‌ها (لیست، مشاهده، حذف)
- 📤 آپلود فایل جدید
- 🔍 جستجوی فایل
- 👥 مدیریت ادمین‌ها (افزودن/حذف)
- 🚫 مدیریت کاربران (بن/آن‌بن)
- 📢 مدیریت عضویت اجباری (افزودن/حذف کانال)
- 💾 بکاپ دیتابیس
- ⚙️ تنظیمات (کانال ذخیره‌سازی)

## 📋 پیش‌نیازها

- Python 3.12+
- pip
- توکن ربات تلگرام از [@BotFather](https://t.me/BotFather)

## 🚀 نصب و اجرا

### 1. کلون پروژه

```bash
cd teriak_uploader
```

### 2. نصب وابستگی‌ها

```bash
pip install -r requirements.txt
```

### 3. تنظیمات

فایل `config.py` را ویرایش کنید:

```python
BOT_TOKEN = "توکن_ربات_شما"
OWNER_ID = 1375809015
STORAGE_CHANNEL_ID = -1001966712871  # کانال ذخیره فایل‌ها
```

### 4. اجرا

```bash
python main.py
```

## ⚙️ تنظیمات مهم

### کانال ذخیره‌سازی (Storage Channel)

1. یک کانال خصوصی در تلگرام ایجاد کنید
2. ربات را در آن کانال ادمین کنید
3. آیدی عددی کانال را در `config.py` در متغیر `STORAGE_CHANNEL_ID` قرار دهید
4. یا از طریق پنل مدیریت > تنظیمات آن را تنظیم کنید

### عضویت اجباری

از طریق پنل مدیریت > عضویت اجباری می‌توانید:
- کانال‌های اجباری را اضافه/حذف کنید
- عضویت اجباری را فعال/غیرفعال کنید

## 🎯 نحوه استفاده

1. ربات را استارت کنید: `/start`
2. ادمین از طریق `/panel` وارد پنل مدیریت می‌شود
3. برای آپلود فایل: پنل > آپلود فایل > ارسال فایل
4. لینک اختصاصی تولید می‌شود: `https://t.me/BOT_USERNAME?start=FILE_UNIQUE_ID`
5. کاربران با باز کردن لینک و استارت ربات، فایل را دریافت می‌کنند

## 📌 نکات

- ربات حتماً باید در کانال ذخیره‌سازی و کانال‌های اجباری **ادمین** باشد
- تلگرام برای ربات‌های غیرپرمیوم محدودیت حجم فایل دارد
- شناسه عددی کانال را می‌توانید از طریق ربات @getidsbot دریافت کنید
